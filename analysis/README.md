# Анализ результатов и эксперименты

Каталог содержит ноутбуки и сводные отчёты по экспериментальной оценке распределённого ансамблевого инференса.

## Подкаталоги

| Подкаталог | Назначение |
|------------|------------|
| `hash_ensemble/` | Эксперименты по распределённому ансамблевому инференсу. Сопоставление одиночных моделей с различным бюджетом кодовой книги и однородных и разнородных ансамблей; сравнение алгоритмов агрегации решений; оценка стабильности ансамблевого голоса. |

## Ансамбли HashedNet

Каталог `hash_ensemble/` содержит:

- ноутбуки `hash_ensemble_kws.ipynb`, `hash_ensemble_kws_v2.ipynb`, `hash_ensemble_ablations.ipynb` с экспериментальной частью;
- сводные результаты в формате JSON (`results_kws.json`, `summary_kws.png`, `results.json`);
- комментарии и заметки по ходу экспериментов в `NOTES.md`.

Ключевые установленные результаты:

- одиночная модель ученика с бюджетом кодовой книги K = 500 даёт точность 90,46 % на тестовой выборке Speech Commands;
- однородный ансамбль из трёх узлов с тем же бюджетом на узел даёт точность 92,69 % при агрегации по среднему логиту;
- среди исследованных алгоритмов агрегации лучший результат показывает `mean_logits` — простое усреднение логитов перед softmax;
- стандартное отклонение точности ансамбля по серии запусков с различным значением `hash_seed` составляет 0,07 п. п. при стандартном отклонении одиночной модели 0,28 п. п., что соответствует снижению дисперсии в 3,84 раза;
- разнородные ансамбли с различными значениями бюджета K на отдельных узлах сохраняют точность в одном диапазоне с однородными ансамблями при сопоставимом суммарном бюджете.

Эти результаты подтверждают применимость распределённого ансамблевого инференса как режима эксплуатации системы, при котором узлы с уникальными значениями `hash_seed` совместно формируют детерминированный коллективный голос.

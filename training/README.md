# Обучение моделей распознавания ключевых слов

Конвейер подготовки данных, обучения и квантования моделей для развёртывания на ESP32-S3. Обучение выполняется во внешней вычислительной среде (облачная среда Google Colab либо локальная машина с графическим ускорителем); конечный результат — бинарное представление модели в формате FlatBuffer для загрузки на узел.

## Конвейер обучения

1. Подготовка набора данных. Используется набор Speech Commands версии 0.02. Применяется десятиклассовая конфигурация: целевые слова `yes`, `no`, `up`, `down`, `left`, `right`, `on`, `off`, `stop`, `go` плюс служебные классы тишины и неизвестного слова.
2. Формирование признаков. Преобразование сырых аудиосэмплов в мел-спектрограммы размерности 49×40 при помощи библиотеки microfrontend. Кэширование признаков на диске для ускорения последующих эпох.
3. Обучение учительской модели. Расширенная архитектура DS-CNN со skip-соединениями и dropout-регуляризацией. Достигаемая точность на тестовой выборке — около 94,8 %.
4. Дистилляция знаний в ученическую модель. Компактная архитектура DS-CNN с сокращённым числом каналов. Функция потерь — взвешенная сумма кросс-энтропии относительно жёстких меток, дистилляционной кросс-энтропии относительно мягких меток учителя и L2-регуляризации. Кэширование логитов учителя для ускорения последующих эпох.
5. Пост-тренировочное INT8-квантование с поканальной шкалой по представительному набору из 200 фрагментов. Конвертация в формат FlatBuffer средствами TFLite Converter.
6. Экспорт квантованной модели в C-массив для встраивания в прошивку.

## Требования

- Python 3.11 и старше.
- TensorFlow версии 2.15 и старше.
- Дополнительные пакеты: `tensorflow-model-optimization`, `numpy`, `librosa`, `tqdm`.
- Облачная среда Google Colab с графическим ускорителем класса T4 либо локальная машина с графическим ускорителем сопоставимого уровня.
